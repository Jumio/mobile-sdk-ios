  • -cancel

    Call the cancel method anytime in the workflow to abort the AuthenticationController process. Dismissing a currently active AuthenticationScanViewController has to be done in the client application. After calling cancel AuthenticationController object has to be newly created. Note: Can only be used when using the Custom-UI functionality, which requires the parameter ‘authenticationScanViewControllerDelegate’ to be set in the AuthenticationConfiguration object.
