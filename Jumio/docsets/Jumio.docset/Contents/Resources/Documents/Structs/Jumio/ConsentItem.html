Jumio Consent Item

    A string containing the identifier of this item

  • url

    A string containing the privacy policy url for this item

  • text

    A string describing the consent item. You need to display this text to the user and the link holder must be clickable and open the privacy url. The link holder will be wrapped by [({})], e.g. “Jumio stores sensitive data. Click [({here})] to get more information.”. Alternatively, you can also display the attributedText, which already contains a clickable link to the privacy url.
